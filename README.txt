ROTI (Return on time invested) MEAN application.

SETUP GUIDE

//Nodejs
1.Create package.json with:
>npm install

2.Install dependencies:
>npm install express body-parser ejs mongojs --save

4.Create server file: server.js

5.If nodemon is not installed
>npm install -g nodemon
Start server with:
>nodemon server

//Angular2
6.Import angular2 app

server name = mood, angular2 app = teammoodapp

7.In the client folder:
>npm start
Compile .ts into .js

//Bower
If not installed, in server folder:
>npm install -g bower
.bowerrc => client/bower_components

8.
>bower install bootstrap --save
>bower install fingerprintjs2 --save
